DC-DC Converter

DC-DC converter — це електронний перетворювач, який змінює один рівень постійної напруги (DC) в інший.

Power STM32 ESP32 Electronics Switching PWM
01 —
Поняття

DC-DC converter — це електронний перетворювач, який змінює один рівень постійної напруги (DC) в інший.

02 —
Типи
  • Buck converter — пониження напруги. 12V → 5V
  • Boost converter — підвищення напруги. 3.7V → 12V
  • Buck-Boost converter — підвищення або пониження напруги
03 —
Використання
  • пониження напруги
  • підвищення напруги
  • стабілізація живлення — підтримка стабільної вихідної напруги, навіть якщо:
    • змінюється вхідна напруга
    • змінюється навантаження
  • живлення різних вузлів системи — одна система часто має кілька різних напруг: 
    12V  → motors
5V   → sensors
3.3V → MCU / ESP32
1.8V → memory / RF
    DC-DC перетворювачі беруть одне джерело живлення і створюють потрібні напруги для кожного вузла.
04 —
Принцип роботи

MOSFET швидко вмикає/вимикає живлення (PWM), дозуючи передачу енергії в індуктор. Індуктор + конденсатор перетворюють імпульси в стабільну DC напругу.

  1. Живлення (Input Voltage)

    Джерело енергії для всієї схеми: 12V, батарея, блок живлення.

  2. Вхідний конденсатор (Input Capacitor)

    IC бере струм ривками, а не плавно. Конденсатор швидко віддає локальну енергію. Згладжує просадки, імпульси струму та шум від перемикання MOSFET.

  3. IC / MOSFET — Switching Element

    Дуже швидко вмикає/вимикає MOSFET і керує PWM (duty cycle). Вимірює зворотній зв'язок (feedback) і змінює D для стабілізації Vout.

    • MOSFET ON: VIN → Inductor → Load — струм іде в дросель, дросель накопичує енергію, навантаження живиться
    • MOSFET OFF: Inductor → Load → Diode — MOSFET розриває ланцюг, струм через індуктор не може миттєво зникнути, тече через діод
  4. Діод — Schottky

    Створює шлях для струму, коли MOSFET OFF. Використовується саме Schottky діод: має низький прямий спад напруги Vf ≈ 0.2–0.4В (звичайний Si-діод — 0.6–0.7В) і майже нульовий час відновлення. Менший Vf → менші втрати → вищий ККД.

  5. Вихідний конденсатор (Output Capacitor)

    Згладжує ripple після дроселя.

  6. Навантаження (Load)

    Споживає стабілізовану напругу: ESP32, STM32, motor driver, sensor

05 —
Частота перемикання (fsw)

Частота, з якою MOSFET вмикається і вимикається. Типовий діапазон — від 100 кГц до 3 МГц.

  • Вища fsw → менший індуктор і конденсатор (менший ripple при тих самих L і C) → менша плата, дешевші компоненти
  • Вища fsw → більше втрат на перемикання в MOSFET → нижчий ККД, більше нагрівання
  • Нижча fsw → вищий ККД, але потрібні більші L і C

06 —
Feedback / Зворотній зв'язок

IC постійно вимірює вихідну напругу і підлаштовує duty cycle щоб тримати Vout стабільним — навіть якщо змінюється навантаження або Vin.

  1. Дільник напруги (R1 / R2)

    Масштабує Vout до опорного рівня IC — зазвичай 0.8В або 1.25В. Вибором R1/R2 задається потрібний Vout.

  2. FB pin

    IC порівнює напругу на FB з внутрішньою опорою (Vref). Якщо FB < Vref → Vout занизька → IC збільшує D. Якщо FB > Vref → зменшує D.

  3. Результат

    Vout залишається стабільним при зміні навантаження або Vin. Це і є регуляція.

Vout = Vref × (1 + R1/R2)   ← формула для більшості IC
