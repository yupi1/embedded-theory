01 — Поняття
Ключова ідея

FSM (кінцевий автомат) — це модель логіки, де система має обмежену кількість станів, перебуває тільки в одному стані одночасно і переходить між станами по подіях/умовах.

03 — Аналогія · Автомат з кавою

Перехід між станами відбувається тільки після конкретної події — натиск кнопки, оплата, завершення приготування. Нижче — інтерактивна симуляція.

Симулятор автомата з кавою — п'ять станів FSM: очікування, вибір, оплата, приготування, видача.

04 — Принцип роботи
  • система стартує в початковому стані
  • чекає вхідну подію (байт, сигнал, таймер)
  • перевіряє поточний стан
  • застосовує умови переходу
  • виконує дію (якщо потрібно)
  • змінює стан або залишається в тому ж
  • повторює цикл для наступної події
05 — Типи FSM
Moore FSM
  • вихід залежить тільки від стану
  • більш стабільний і передбачуваний
  • часто використовується для основної логіки процесів
Mealy FSM
  • вихід залежить від стану + вхідної події
  • швидша реакція на input
  • часто використовується для UI та обробки подій
Формально

Moore і Mealy — формальні математичні моделі FSM.

МодельФормулаЗастосування
Moore output = f(state) Основні стани процесу
Mealy output = f(state, input) Реакція на input, UI, обробка подій
На практиці

Системи зазвичай гібридні: Moore — для основних станів процесу, Mealy — для реакції на input.

06 — Код · C · switch/case
C · FSM coffee_machine.c 
typedef enum {
    WAIT_START, WAIT_SELECTION, WAIT_PAYMENT, WAIT_BREW, DELIVER
} state_t;

typedef enum {
    NO_EVENT, CLIENT_DETECTED, BUTTON_PRESSED, PAYMENT_DONE, BREW_DONE
} event_t;

void process_event(state_t *state, event_t event) {
    switch (*state) {

    case WAIT_START:
        if (event == CLIENT_DETECTED) {
            *state = WAIT_SELECTION;
        }
        break;

    case WAIT_SELECTION:
        if (event == BUTTON_PRESSED) {
            *state = WAIT_PAYMENT;
        }
        break;

    case WAIT_PAYMENT:
        if (event == PAYMENT_DONE) {
            *state = WAIT_BREW;
        }
        break;

    case WAIT_BREW:
        if (event == BREW_DONE) {
            *state = DELIVER;
        }
        break;

    case DELIVER:
        printf("Coffee delivered\n");
        *state = WAIT_START;
        break;
    }
}
